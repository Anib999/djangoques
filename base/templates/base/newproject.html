Use these commands to start a new project on django
  1. First create a virtual environment
  2. virtualenv [envname]
  3. start the virtual environment
  4. env\Scripts\activate
  5. Note*: env = [envname]
  1. Then we can install django
  2. pip install django
  3. after installing django we can start project of admin
  4. django-admin startproject [projectname]
  5. We can start coding here but as we can seperate apps
  1. We can then start our app
  2. python manage.py startapp [appname]

With this we can create our django app successfully. Now we can use python manage.py runserver.

Now after our app hase been created we need to keep our [appname].apps.BaseConfig in INSTALLED_APPS so it can be used.

And if we have templates outside of app then we need to define it in templates DIRS as Base_dir / 'templates'

For static files use following in settings.py

STATIC_URL = 'static/' STATICFILES_DIRS = [ BASE_DIR / 'static' ] MEDIA_URL = '/images/' MEDIA_ROOT = BASE_DIR / 'static/images'

now after all things are set up. use "python manage.py migrate" to migrate unappiled migrations

python manage.py makemigrations to finally migrate to db

python manage.py createsuperuser to create a super user for login in django built in admin panel
