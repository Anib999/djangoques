Use these commands to start a new project on django
  1. First create a virtual environment
  2. virtualenv [envname]
  3. start the virtual environment
  4. env\Scripts\activate
  5. Note*: env = [envname]
  1. Then we can install django
  2. pip install django
  3. after installing django we can start project of admin
  4. django-admin startproject [projectname]
  5. We can start coding here but as we can seperate apps
  1. We can then start our app
  2. python manage.py startapp [appname]

With this we can create our django app successfully. Now we can use python manage.py runserver.
